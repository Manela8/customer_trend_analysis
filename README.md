# 👨🏻‍💻 Customer Behavior Data Analyst Portfolio Project

A complete, industry-standard, end-to-end data analytics workflow designed to mirror the real responsibilities of professional analysts in modern business environments — covering everything from data preparation and modeling to insight generation, visualization, and reporting.

---

## 📌 Project Overview

This project simulates a corporate-grade analytics workflow, demonstrating the ability to translate raw data into strategic business intelligence.

| Stage | Tool | Description |
|---|---|---|
| 🐍 Data Preparation & EDA | Python | Clean and transform raw dataset |
| 🗄️ Data Analysis | SQL (MySQL) | Extract insights on customer segments, loyalty, and purchase drivers |
| 📊 Visualization | Power BI | Build interactive dashboards for stakeholders |

---

## 🛠️ Tech Stack

- **Python** — pandas, SQLAlchemy, PyMySQL
- **MySQL** — database, SQL queries, CTEs, window functions
- **Power BI** — interactive dashboards
- **Jupyter Notebook** — EDA and data pipeline
